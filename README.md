# End Task All

أداة لويندوز تساعد المستخدمين اللي جهازهم بيتعلق (Freeze) أو بيهنج أو بيبطأ بسبب برنامج أو أكتر شغالين في نفس الوقت. بضغطة اختصار كيبورد واحدة (Ctrl+Alt+A)، بيتقفلوا كل البرامج المفتوحة فورًا.

## ⚠️ تحذير قبل الاستخدام
الأداة دي بتقفل **كل البرامج المفتوحة بالإجبار** (Force Close) ما عدا:
- مستكشف الملفات (Explorer)
- مدير سطح المكتب (DWM)

**أي بيانات غير محفوظة في البرامج المفتوحة (زي مستند Word أو Excel لسه ماتحفظش) هتضيع فورًا.**
استخدمها فقط لما الجهاز يكون عالق فعلاً ومحتاج تحرير سريع للموارد.

## طريقة التثبيت (مرة واحدة بس)

1. حمّل المشروع كامل (Code → Download ZIP) وفك الضغط
2. تأكد إن ملفي `Install.bat` و `CreateShortcut.ps1` موجودين في **نفس المجلد**
3. دبل كليك على `Install.bat` لتشغيله (هيطلب صلاحيات، اضغط Yes)
4. هيتعمل تلقائيًا اختصار جديد على سطح المكتب اسمه **"End Task All"**

## طريقة الاستخدام اليومية

لما الجهاز يهنج أو يبطأ:
1. دوس **Ctrl + Alt + A** من الكيبورد (مش محتاج ماوس)
2. هتظهرلك نافذة تأكيد من ويندوز (User Account Control) بتوضح إن العملية جايه من `taskkill.exe` الموقّع من Microsoft
3. دوس **Enter** أو اضغط **Yes**
4. كل البرامج المفتوحة هتتقفل فورًا

## ملف بديل: EndTaskAll.bat

لو مش عايز تعمل اختصار كيبورد وعايز بس ملف تشغّله يدوي كل مرة، فيه ملف `EndTaskAll.bat` بديل بيسألك تأكيد كتابي (Y) قبل التنفيذ، مناسب لو الجهاز لسه شغال بشكل طبيعي ومش هانج تمامًا.

## طريقة عمله (تقنيًا)

السكريبت بيستخدم أداة `taskkill` المدمجة في ويندوز مع فلاتر (Filters) تحدد:
- بس البرامج الشغالة تحت المستخدم الحالي
- استثناء العمليات الأساسية اللي محتاجينها عشان الجهاز يفضل شغال بشكل طبيعي

```batch
taskkill /F /FI "USERNAME eq %USERNAME%" /FI "IMAGENAME ne explorer.exe" /FI "IMAGENAME ne dwm.exe" /FI "IMAGENAME ne cmd.exe" /FI "IMAGENAME ne taskmgr.exe"
```

## متطلبات التشغيل
- ويندوز 7 أو أحدث
- مفيش حاجة تانية محتاج تنزلها، كل الأدوات المستخدمة مدمجة في ويندوز أصلاً
